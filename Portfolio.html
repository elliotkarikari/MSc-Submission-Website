This project uses Agent Based Models to understand sheep behaviour in an environment. 20 object have been given agency. The sheep move, eat, store what has been eaten, share, and place excess store into their environment, causing it to grow. Watch out for the wolf.

Object-Oriented Programming allows code to be segeragated and perform multiple functions simultaneously.This model iterates 500 times per RUN.
